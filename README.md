# Mortgage-Calculator-In-Java
A simple Mortgage calculator in java.

This is a simple project where I build using NumberFormat, learn the style of Java, type conversions, using Scanner to read from user input and ensure there are no errors in my fundamentals. 
It is quite simple and uses the formula : M = P × [ r(1+r)ⁿ / (1+r)ⁿ - 1 ]
